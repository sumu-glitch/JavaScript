JavaScript DataTypes

There are two types of datatypes in JavaScript:

1. Primitive

Primitive datatypes are the basic building blocks of data in JavaScript. They include:

  1. String

    Ex: let name = "John";

  2. BigInt

    Ex: let bigNumber = 1234567890n;

  3. Number

    Ex: let a = Number(6);

  4. Boolean

    Ex: let a = true;

  5. Null

    Ex: let a = null;

  6. Undefined

    Ex: let a = undefined;

  7. Symbol

    Ex: let a = Symbol(6);

2. Non-Primitive

Non-primitive datatypes are more complex:

  1. Object

    Ex: let obj = { name: "John", age: 30 };

  2. Array

    Ex: let arr = [1, 2, 3];

  3. Function

    Ex: function greet() { console.log("Hello"); }

  4. Date

    Ex: let currentDate = new Date();

  5. RegExp

    Ex: let regex = /ab+c/;

  6. Error

    Ex: let error = new Error("Oops!");

  7. Map

    Ex: let map = new Map();

  8. Set

    Ex: let set = new Set();

JavaScript is a dynamic language. You don't need to define variable types explicitly like in TypeScript or Java.
