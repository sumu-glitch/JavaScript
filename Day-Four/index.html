JavaScript DataTypes

There are two types of datatypes

1.Primitive

2.Non-Primitive

1.Primitive

Primitive datatypes are the basic building blocks of data in JavaScript. They include:

  • String

    • Ex:- let name = "John";

  • BigInt

    • Ex:- let bigNumber = 1234567890123456789012345678901234567890n;

  • Number

    • Ex:- let a = Number(6)

  • Boolean

    • Ex:- let a = true

  • Null

    • Ex:- let a = Null

  • Undefined

    • Ex:- let a = Undefined

  • Symbol (ES6)

    • Ex:- let a = Symbol(6)

2.Non-Primitive

Non-primitive datatypes are more complex and can hold collections of values or more complex entities. They include:

  • Object

    • Ex:- let obj ={ name: "John", age: 30, isStudent: false };

  • Array

    • Ex:- let arr = [1, 2, 3, 4, 5];

  • Function

    • Ex:- function greet() { console.log("Hello, World!"); }


    Othe type of non-primitive DataTypes

  • Date

    • Ex:- let currentDate = new Date();

  • RegExp

    • Ex:- let regex = /ab+c/;

  • Error

    • Ex:- let error = new Error("Something went wrong!");

  • Map

    • Ex:- let map = new Map(); map.set("key1", "value1"); map.set("key2", "value2");

  • Set

    • Ex:- let set = new Set(); set.add(1); set.add(2); set.add(3);

JavaScript is a dynamic language and not static

which means that variables can hold values of different types during runtime. Unlike languages such as Typescript or Java, you don't need to declare the data type of a variable explicitly.
